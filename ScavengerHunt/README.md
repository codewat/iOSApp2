Scavenger Hunt App (iOSApp2)
This is a mobile application developed as part of an iOS Development assignment. The app allows users to participate in a local scavenger hunt, providing clues for 10 items and offering discount rewards based on the number of items found.

Features
Scavenger Item List: A list view displaying all 10 hunt items.

Progress Tracking: Real-time calculation of found items and corresponding discount codes.

Toggle Status: Users can mark items as found or unmark them if they made a mistake.

MVVM Architecture: Built using the Model-View-ViewModel pattern for clean and maintainable code.

Technologies Used
Language: Swift

Framework: SwiftUI

Architecture: MVVM (Model-View-ViewModel)

Tools: Xcode, Git, GitHub

How to Run
Clone this repository to your local machine.

Open the project in Xcode.

Select a Simulator (e.g., iPhone 15 or newer).

Press the Run button (Command + R).
